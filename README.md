# JAMAL AL-SHUWAIKH - Tailor Management System

Professional management software for JAMAL AL-SHUWAIKH tailor shop.

## Features
- Client management with measurements
- Order tracking with status updates
- Invoice generation with QR codes
- PDF export and printing
- WhatsApp sharing
- Expense tracking
- Advanced search functionality
- Data backup and restore

## Setup
1. Upload all files to GitHub
2. Enable GitHub Pages
3. Open the provided URL
4. Start using immediately

## Usage
- Use top buttons to switch between functions
- Search clients by name, phone, or book number
- Create invoices with multiple items
- Generate QR codes for payments
- Export PDFs and share via WhatsApp
